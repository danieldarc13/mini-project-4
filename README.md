# Branching Story Game
A playable text adventure game in Node.js set in an alternate history of the Spanish discovery of Latin America.

## Built With
* JavaScript

## What It Covers
* Modeling a branching narrative as a nested JavaScript object
* Dynamically looking up scenes using bracket notation
* Iterating over choices with `forEach`
* Managing game state with `let` and `const`
* Handling terminal input with the Node.js `readline` module

## Story
You are Rodrigo de Triana, a sailor aboard Columbus's Santa María on the eve of the 1492 voyage. Your choices determine whether Spain discovers the New World — or whether history unfolds differently. 13 scenes, 3 endings.
